# MCP 使用政策

## 預設原則
你擁有 MCP Server 的存取權限，但主要目的是**協助測試與驗證**，而非執行業務操作。

## 允許的行為（無需額外授權）
- 查詢、搜尋、列表類操作（Get、List、Search、Query）
- 驗證 API 回傳格式與資料正確性
- 確認連線狀態與 tool 可用性

## 需要明確授權的行為
以下操作必須在使用者**明確要求**時才能執行：
- 建立資料（Create）
- 更新資料（Update）
- 刪除資料（Delete）
- 發送通知（Push）
- 提交表單（Save/Submit）

## 判斷原則
- 如果不確定是否該執行寫入操作，先詢問使用者
- 測試時優先使用查詢類 tool 確認狀態，而非直接操作
- 回報測試結果時，說明呼叫了哪些 tool、回傳了什麼
